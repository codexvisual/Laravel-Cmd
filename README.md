## 🖥 WINDOWS COMMANDS

⚡ System Info  
```powershell
systeminfo
winver
```

🧹 Cleanup & Speed  
```powershell
cleanmgr
del /q/f/s %TEMP%\*
taskmgr
```

🌐 Network Tools  
```powershell
ipconfig
ipconfig /flushdns
ping google.com
```

🔐 Security  
```powershell
windowsdefender:
Start-MpScan -ScanType QuickScan
```

## ⚙️ LARAVEL COMMANDS

🚀 Setup  
```bash
composer create-project laravel/laravel app
php artisan serve
```

🗄 Database  
```bash
php artisan migrate
php artisan migrate:fresh
```

🧱 Build  
```bash
php artisan make:model Product -m
php artisan make:controller ProductController
```

⚡ Optimization  
```bash
php artisan optimize:clear
php artisan config:cache
```
